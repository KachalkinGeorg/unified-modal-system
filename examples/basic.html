🧩 Unified Modal System (UM)

Режим работы: jQuery UI: Выключен | Custom (Vanilla): Встроен
💡 Система сама определяет библиотеки. Достаточно указать window.UM_CONFIG.jquery_ui = true, если jQuery UI подключён или window.UM_CONFIG.jquery_ui = false если отключен.

1. Уведомления UM_Alert(type, message, options)

2. Модальные окна UM_ShowModal(opts)

📖 Как подключить в реальном проекте

<!-- 1. Конфиг и языки (до um-core.js) -->
<script>
  window.UM_CONFIG = { jquery_ui: false };
  var lang = { close: 'Закрыть', save: 'Сохранить', cancel: 'Отмена' };
</script>

<!-- 2. Основной файл системы -->
<script src="/js/lib/um-core.js"></script>

<!-- 3. Использование -->
<script>
  UM_Alert('success', 'Готово!');
  UM_ShowModal({ url: '/ajax/data.php', title: 'Данные' });
</script>
